对抗攻击：PGD 攻击原理、公式推导与代码实现

最后更新于2026年6月12日 下午

PGD（Projected Gradient Descent）是目前最强的一阶对抗攻击方法，由 Madry 等人在 2018 年提出。本文从原理出发，推导完整公式，并用代码演示攻击流程。

1. 对抗样本的定义

给定分类器 fθf_\thetafθ​、输入 xxx、真实标签 yyy，对抗样本 xadvx^{adv}xadv 满足：

fθ(xadv)≠y且∥xadv−x∥p≤ϵf_\theta(x^{adv}) \neq y \quad \text{且} \quad \|x^{adv} - x\|_p \leq \epsilon fθ​(xadv)=y且∥xadv−x∥p​≤ϵ

其中 ϵ\epsilonϵ 是扰动预算，∥⋅∥p\|\cdot\|_p∥⋅∥p​ 通常取 L∞L_\inftyL∞​ 或 L2L_2L2​ 范数。

直觉： 在人眼无法察觉的微小扰动下，让模型预测出错。

2. 从 FGSM 到 PGD

2.1 FGSM（单步攻击）

Goodfellow 等人提出的 Fast Gradient Sign Method：

xadv=x+ϵ⋅sign(∇xL(θ,x,y))x^{adv} = x + \epsilon \cdot \text{sign}(\nabla_x \mathcal{L}(\theta, x, y)) xadv=x+ϵ⋅sign(∇x​L(θ,x,y))

问题： 单步攻击容易被模型的非线性"绕过"，攻击成功率有限。

2.2 PGD（多步投影梯度下降）

PGD 将对抗攻击建模为约束优化问题：

max⁡xadv∈B(x,ϵ)L(θ,xadv,y)\max_{x^{adv} \in \mathcal{B}(x, \epsilon)} \mathcal{L}(\theta, x^{adv}, y) xadv∈B(x,ϵ)max​L(θ,xadv,y)

其中 B(x,ϵ)={x′:∥x′−x∥∞≤ϵ}\mathcal{B}(x, \epsilon) = \{x' : \|x' - x\|_\infty \leq \epsilon\}B(x,ϵ)={x′:∥x′−x∥∞​≤ϵ} 是以 xxx 为中心、半径 ϵ\epsilonϵ 的 L∞L_\inftyL∞​ 球。

3. PGD 公式推导

3.1 迭代更新规则

PGD 用梯度上升（最大化损失）+ 投影（保持在约束集内）交替进行：

第 ttt 步更新：

xt+1=ΠB(x,ϵ)(xt+α⋅sign(∇xtL(θ,xt,y)))x^{t+1} = \Pi_{\mathcal{B}(x,\epsilon)} \left( x^t + \alpha \cdot \text{sign}(\nabla_{x^t} \mathcal{L}(\theta, x^t, y)) \right) xt+1=ΠB(x,ϵ)​(xt+α⋅sign(∇xt​L(θ,xt,y)))

  • α\alphaα：步长（通常取 ϵ/T\epsilon / Tϵ/T 或固定小值）
  • TTT：总迭代步数
  • ΠB\Pi_{\mathcal{B}}ΠB​：投影算子

3.2 L∞L_\inftyL∞​ 投影算子

将扰动 δ=xadv−x\delta = x^{adv} - xδ=xadv−x 投影回 [−ϵ,ϵ][-\epsilon, \epsilon][−ϵ,ϵ]：

ΠB(x,ϵ)(x′)=x+clip(x′−x, −ϵ, ϵ)\Pi_{\mathcal{B}(x,\epsilon)}(x') = x + \text{clip}(x' - x,\ -\epsilon,\ \epsilon) ΠB(x,ϵ)​(x′)=x+clip(x′−x, −ϵ, ϵ)

等价地，对每个像素维度 iii：

xiadv=clip(xi′, xi−ϵ, xi+ϵ)x^{adv}_i = \text{clip}(x'_i,\ x_i - \epsilon,\ x_i + \epsilon) xiadv​=clip(xi′​, xi​−ϵ, xi​+ϵ)

3.3 完整算法

输入：x, y, ϵ, α, Tx0←x+U(−ϵ,ϵ)（随机初始化）for t=0,1,…,T−1:gt←∇xtL(θ,xt,y)xt+1←ΠB(x,ϵ)(xt+α⋅sign(gt))return xT\begin{aligned} &\textbf{输入：} x,\ y,\ \epsilon,\ \alpha,\ T \\ &x^0 \leftarrow x + \mathcal{U}(-\epsilon, \epsilon) \quad \text{（随机初始化）} \\ &\textbf{for } t = 0, 1, \ldots, T-1: \\ &\quad g^t \leftarrow \nabla_{x^t} \mathcal{L}(\theta, x^t, y) \\ &\quad x^{t+1} \leftarrow \Pi_{\mathcal{B}(x,\epsilon)}\left(x^t + \alpha \cdot \text{sign}(g^t)\right) \\ &\textbf{return } x^T \end{aligned} ​输入：x, y, ϵ, α, Tx0←x+U(−ϵ,ϵ)（随机初始化）for t=0,1,…,T−1:gt←∇xt​L(θ,xt,y)xt+1←ΠB(x,ϵ)​(xt+α⋅sign(gt))return xT​

随机初始化的作用： 避免从 xxx 出发陷入局部最优，使攻击更强。

5. 关键超参数分析

参数 典型值 作用
ϵ\epsilonϵ 8/255（L∞L_\inftyL∞​） 扰动预算，越大攻击越强但越明显
α\alphaα 2/255 步长，过大会震荡，过小收敛慢
TTT 20~40 迭代步数，越多越强，计算越慢
随机重启 5~10 次 多次随机初始化取最强攻击

经验规则： α=ϵ/(T/4)\alpha = \epsilon / (T/4)α=ϵ/(T/4)，步长约为预算的 4/T4/T4/T。

6. PGD 与对抗训练

PGD 最重要的应用是 Madry 对抗训练，将对抗样本加入训练：

min⁡θE(x,y)∼D[max⁡δ∈B(0,ϵ)L(θ,x+δ,y)]\min_\theta \mathbb{E}_{(x,y) \sim \mathcal{D}} \left[ \max_{\delta \in \mathcal{B}(0,\epsilon)} \mathcal{L}(\theta, x+\delta, y) \right] θmin​E(x,y)∼D​[δ∈B(0,ϵ)max​L(θ,x+δ,y)]

内层 max⁡\maxmax 用 PGD 求解，外层 min⁡\minmin 用 SGD 更新参数。这是目前最有效的对抗鲁棒性训练方法之一。

7. 小结

  • PGD = 多步 FGSM + 投影 + 随机初始化
  • 核心公式：xt+1=ΠB(xt+α⋅sign(∇L))x^{t+1} = \Pi_{\mathcal{B}}\left(x^t + \alpha \cdot \text{sign}(\nabla \mathcal{L})\right)xt+1=ΠB​(xt+α⋅sign(∇L))
  • 攻击强度随步数 TTT 增加而趋于饱和
  • 是对抗训练（Madry AT）的标准内层求解器
