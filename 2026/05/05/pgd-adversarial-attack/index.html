PGD 对抗攻击：原理、公式推导与代码实现

最后更新于2026年6月12日 下午

PGD（Projected Gradient Descent）是目前最强的一阶对抗攻击方法，也是对抗训练的标准攻击基准。本文从原理出发，完整推导公式并给出可运行代码。

1. 什么是对抗攻击？

给定一个分类器 fθf_\thetafθ​，对于输入图片 xxx 和真实标签 yyy，对抗攻击的目标是找到一个人眼不可见的扰动 δ\deltaδ，使得模型预测错误：

fθ(x+δ)≠yf_\theta(x + \delta) \neq y fθ​(x+δ)=y

同时扰动受到约束，通常用 ℓ∞\ell_\inftyℓ∞​ 范数限制：

∥δ∥∞≤ϵ\|\delta\|_\infty \leq \epsilon ∥δ∥∞​≤ϵ

其中 ϵ\epsilonϵ 通常取 8/2558/2558/255（图像像素值归一化到 [0,1][0,1][0,1]）。

2. FGSM：PGD 的前身

FGSM（Fast Gradient Sign Method，Goodfellow 2014）是最简单的对抗攻击，一步完成：

xadv=x+ϵ⋅sign(∇xL(fθ(x),y))x^{adv} = x + \epsilon \cdot \text{sign}(\nabla_x \mathcal{L}(f_\theta(x), y)) xadv=x+ϵ⋅sign(∇x​L(fθ​(x),y))

直觉：沿着损失函数对输入的梯度方向走一步，让损失最大化。

FGSM 的缺点是只走一步，攻击强度有限。PGD 将其扩展为多步迭代。

3. PGD 攻击：公式推导

3.1 优化目标

PGD 将对抗攻击建模为一个约束优化问题：

max⁡δ∈SL(fθ(x+δ),y)\max_{\delta \in \mathcal{S}} \mathcal{L}(f_\theta(x + \delta), y) δ∈Smax​L(fθ​(x+δ),y)

其中约束集合 S={δ:∥δ∥∞≤ϵ}\mathcal{S} = \{\delta : \|\delta\|_\infty \leq \epsilon\}S={δ:∥δ∥∞​≤ϵ} 是以原始输入为中心的 ℓ∞\ell_\inftyℓ∞​ 球。

3.2 迭代更新规则

PGD 用投影梯度上升求解，第 ttt 步更新为：

δt+1=ΠS(δt+α⋅sign(∇δL(fθ(x+δt),y)))\delta^{t+1} = \Pi_{\mathcal{S}}\left(\delta^t + \alpha \cdot \text{sign}(\nabla_\delta \mathcal{L}(f_\theta(x + \delta^t), y))\right) δt+1=ΠS​(δt+α⋅sign(∇δ​L(fθ​(x+δt),y)))

其中：

  • α\alphaα 是每步的步长（通常取 ϵ/4\epsilon / 4ϵ/4 或 2/2552/2552/255）
  • ΠS\Pi_{\mathcal{S}}ΠS​ 是投影算子，将扰动裁剪回合法范围
  • sign(⋅)\text{sign}(\cdot)sign(⋅) 取梯度的符号

3.3 投影算子

对于 ℓ∞\ell_\inftyℓ∞​ 约束，投影就是简单的裁剪：

ΠS(δ)=clip(δ,−ϵ,ϵ)\Pi_{\mathcal{S}}(\delta) = \text{clip}(\delta, -\epsilon, \epsilon) ΠS​(δ)=clip(δ,−ϵ,ϵ)

同时还需要保证对抗样本的像素值在合法范围内：

xadv=clip(x+δ,0,1)x^{adv} = \text{clip}(x + \delta, 0, 1) xadv=clip(x+δ,0,1)

3.4 随机初始化

PGD 的关键改进（相比 FGSM）是随机初始化扰动：

δ0∼Uniform(−ϵ,ϵ)\delta^0 \sim \text{Uniform}(-\epsilon, \epsilon) δ0∼Uniform(−ϵ,ϵ)

这使得 PGD 能探索约束集合内的不同区域，避免陷入局部最优，攻击更强。

7. PGD 与对抗训练

PGD 攻击最重要的应用是 Madry 对抗训练（2018）：

min⁡θE(x,y)[max⁡δ∈SL(fθ(x+δ),y)]\min_\theta \mathbb{E}_{(x,y)}\left[\max_{\delta \in \mathcal{S}} \mathcal{L}(f_\theta(x+\delta), y)\right] θmin​E(x,y)​[δ∈Smax​L(fθ​(x+δ),y)]

内层 max⁡\maxmax 用 PGD 求解，外层 min⁡\minmin 用 SGD 更新模型参数。这是目前最有效的对抗防御方法之一。

总结

PGD 攻击的核心要点：

  1. 本质：在 ℓ∞\ell_\inftyℓ∞​ 球约束下最大化损失的投影梯度上升
  2. 关键：随机初始化 + 多步迭代，比 FGSM 强得多
  3. 应用：既是攻击工具，也是对抗训练的标准内层求解器
  4. 参数：ϵ=8/255\epsilon=8/255ϵ=8/255，α=2/255\alpha=2/255α=2/255，K=20K=20K=20 是图像分类的常用设置
