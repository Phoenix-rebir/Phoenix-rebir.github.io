必踩坑总结

坑1：H100 不支持旧 PyTorch

H100 架构为 sm_90，PyTorch 2.0 以下不包含该架构编译，运行时会报 CUDA error。必须换 PyTorch 2.1+。

坑2：GitHub 无法访问

部分服务器无法访问 GitHub，pip install git+https://... 会超时。解决方案：在有网络的机器下载 tar.gz，传到服务器后本地安装。
